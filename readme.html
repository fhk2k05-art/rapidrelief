🚨 RapidRelief

Smart Emergency Response System for faster help using nearby people and real-time tracking.

🌟 Features

  • 🚑 Nearest hospital, police, fire, mechanic navigation
  • 👥 Nearby people detection within 5km
  • 🚨 SOS with auto alert system
  • 📡 Live location tracking
  • 🗺️ Interactive map with real-time updates

⚙️ How it works

  • User selects emergency type
  • Clicks Call or SOS
  • System finds nearest help
  • Nearby users are alerted
  • Admin can monitor activity

🛠️ Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express
  • Maps: Leaflet.js

🚀 Future Improvements

  • Real-time live map
  • Push notifications
  • Mobile app (APK)
  • AI emergency detection
